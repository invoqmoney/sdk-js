# `@invoq/server`

Node.js SDK for invoq server APIs and webhook verification — create stablecoin invoices and fulfill orders with signed webhooks. [invoq](https://invoq.money) is non-custodial: funds settle straight to your own wallet.

Install:

```sh
npm install @invoq/server
```

Requires Node.js 20 or newer.

For setup, examples, and API usage, see the main [invoq JavaScript SDKs README](https://github.com/invoqmoney/sdk-js#readme).
