# `@invoq/checkout`

Browser SDK for opening the [invoq](https://invoq.money) stablecoin checkout — an in-page payment modal that customers never have to leave. Framework-agnostic, zero runtime dependencies, TypeScript types included.

Install:

```sh
npm install @invoq/checkout
```

For setup, examples, and API usage, see the main [invoq JavaScript SDKs README](https://github.com/invoqmoney/sdk-js#readme).
